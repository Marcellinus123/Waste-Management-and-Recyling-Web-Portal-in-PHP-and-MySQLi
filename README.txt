INSTALLATION INSTRUCTIONS
=========================
=========================
NOTE: You must copy the 'waste' folder into C:\xampp\htdocs\
1. Start Xampp Server
2. Go to xamp control panel and start Apache, Mysql
3. Click on the 'admin' on the mysql section and goto phpmyadmin page
http://localhost/phpmyadmin/
4. Create a database named 'waste_wizard_db' if not exist
5. Go to 'Import' in phpmyadmin  and click on 'choose file'
6. In your project folder, locate 'waste_wizard_db.sql' file and import it.
7. Run the software by visiting http://localhost/waste

REQUIREMENTS
!!!!!!!!!!!!!!!!!!!!!!!
1. PHP 7.2 + supported
